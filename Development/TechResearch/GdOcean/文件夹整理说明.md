# GdOcean 文件夹整理说明

## 📁 当前结构

```
GdOcean/
├── src/              # C++ 源代码
├── godot-cpp/        # Godot C++ 绑定（子模块）
├── bin/              # 构建产物（应忽略）
├── *.tscn            # 场景文件
├── *.gd              # GDScript 测试脚本
├── *.gdextension     # GDExtension 配置
├── SConstruct        # 构建脚本
├── build*.log        # 构建日志（应清理）
└── README.md         # 说明文档
```

## 🧹 建议清理的文件

### 构建日志（可删除）
- `build*.log`
- `build*.txt`
- `error*.log`
- `*.log`

这些文件可以通过 `.gitignore` 忽略，或移动到 `build_logs/` 文件夹。

### 构建产物（应忽略）
- `bin/` 文件夹
- `*.pdb`, `*.exp`, `*.lib`, `*.obj`

## ✅ 已修复的问题

1. **节点路径错误**
   - 修复：`HybridOceanTest.tscn` 中的绝对路径改为相对路径
   - 修复：`OceanTest.gd` 中使用 `get_path_to()` 获取相对路径

2. **变量遮蔽警告**
   - 修复：`local_ocean` 重命名为 `local_ocean_sim`

3. **未使用参数警告**
   - 修复：`_process(delta)` → `_process(_delta: float)`

## 📝 建议的文件夹结构

GdOcean/
