# 專案 "Corsair" (海盜)：玩法與派系設計
> **設計理念**：觸感物理 (Tactile Physics)、模組化工程、動態生態系統。

## 1. 願景與核心支柱
**定位**：一款定義性的「觸感物理」開放世界海盜遊戲。填補《盜賊之海》（冒險）與《圍攻》（建造）之間的市場空白。
*   **物理優先**：拒絕「按 F 鍵互動」。裝填砲彈、調整帆角、搬運貨物都必須手動且符合物理慣性。
*   **模組化工程**：船隻由具有質量、浮力、耐久度的獨立模組組裝而成，而非單一模型。
*   **無縫垂直世界**：從深海遺跡延伸至萬米高空的天空島嶼。
*   **動態派系**：一個基於資源與影響力圖譜的活體世界，而非靜態的敵對 NPC。

---

## 2. 派系與經濟
世界被劃分為由三大派系控制的「貿易區」。經濟驅動戰爭（例如：缺糧會導致甚至引發入侵）。

### 2.1 鋼鐵兄弟會 (The Iron Syndicate)
*   **主題**：工業、秩序、柴油龐克。
*   **視覺**：鏽蝕金屬、鉚釘、黑煙、橙色警示燈。
*   **船隻原型**：明輪無畏艦 (Paddlewheel Dreadnoughts)。
*   **動力**：**蒸汽鍋爐**。無視風向，推力強大，但消耗**煤炭**且有過熱爆炸風險。
*   **戰術**：側舷重甲抗線（T字戰術），利用重型火炮齊射。
*   **特殊模組**：**液壓撞角** (Hydraulic Ram)。物理驅動，造成 500% 結構破壞。

### 2.2 漂流木公約 (The Driftwood Covenant)
*   **主題**：拾荒、混亂、極致適應。
*   **視覺**：拼湊木板、彩色布料、骨頭裝飾、漁網。
*   **船隻原型**：雙體 / 三體快船 (Catamarans)。
*   **動力**：**三角帆** (Lateen Sails)。極高機動性，逆風航行能力強。
*   **戰術**：狼群戰術。利用速度環繞大船，發射鏈彈破壞桅杆。
*   **特殊模組**：**魚叉絞盤** (Harpoon Winch)。帶有物理繩索，可拖拽敵船或進行強制接舷。

### 2.3 深淵教團 (The Tidebound)
*   **主題**：神秘、深海、生物科技。
*   **視覺**：藤壺、半透明生物玻璃、發光組織、石質船殼。
*   **船隻原型**：半潛式生物艦 (Semi-submersible Bio-ships)。
*   **動力**：**鰭肢推進**。無聲、無航跡。
*   **戰術**：垂直維度打擊。下潛至水下規避砲火，從目標船底上浮衝撞。
*   **特殊模組**：**壓載氣囊** (Ballast Bladder)。手動控制浮力，實現潛水艇玩法。

## 3. 核心玩法機制

### 3.1 「甲板手」循環 (The Deck Hand Loop)
操作是物理小遊戲，而非讀取進度條。
*   **裝填火砲**：去火藥庫抱火藥桶 -> 塞入砲口 -> 抱砲彈塞入 -> 用推桿壓實。在槍林彈雨中完成這些動作創造了巨大的張力。
*   **損管機制**：破洞是物理性的。水會增加船體質量 -> 降低浮力 -> 導致下沉。玩家必須搬運木板，用錘子敲擊（節奏遊戲）封堵，並手動舀水。

### 3.2 邏輯系統 (海洋紅石)
允許玩家對船隻進行邏輯編程。
*   **元件**：傳感器（水深、敵艦探測、風向）& 執行器（活塞、馬達、點火器）。
*   **範例**：
    *   *自動排水*：水位傳感器 > 0.5m -> 啟動 蒸汽泵。
    *   *姿態穩定*：陀螺儀傾角 > 10° -> 伸出 反向配重活塞。
    *   *防登船陷阱*：壓力板檢測到非船員重量 -> 觸發 彈射板。

### 3.3 物理異常 (世界事件)
*   **大漩渦 (The Maelstrom)**：具有強大引力場的巨型漏斗，考驗操舵與離心力運用。
*   **重力反轉區 (Gravity Wells)**：天空遺跡下方，重力減弱至 0.2g。船隻會浮空，砲彈彈道變平。
*   **幽靈船重播**：將其他玩家死亡時的物理狀態實例化為半透明幽靈船，標記沉船寶藏位置。

## 4. 造船工程概念
*   **動態質心**：在左側安裝重砲，必須在右側放置壓艙石平衡，否則船會側傾。
*   **結構完整性**：船隻是圖結構 (Graph)。如果不與「龍骨」(Root) 連接，模組會物理斷裂脫落。
*   **氣動力學**：帆是翼型 (Airfoil)，產生升力而非單純推力。玩家需要根據風向調整攻角 (Trimming)。
